Import Carriers from Excel

Instructions:

Upload an Excel file (.xlsx or .xls) with the following columns:

  • Code - Carrier code (e.g., "ACE-001")
  • Name - Carrier name (e.g., "ACE Insurance Company")
  • RepEmail - Representative email (e.g., "john.doe@carrier.com")

Note: The first row should contain column headers.

Email Integration

⚫ Not Connected
Connected as:

Connect your Outlook account to send and receive emails directly from this application. This will allow you to:

  • Send emails directly through your Outlook account
  • Receive replies from carriers
  • Automatically sync email conversations
  • Attach files from your computer

Note: You'll be redirected to Microsoft to authorize this application.

Tip: If sync isn't finding your emails, try clicking "Reset Sync Date" then "Sync Emails" again.

Recommended: Use a public URL so your logo displays correctly in sent emails.

Where to get a public URL?
  • If deployed to Firebase Hosting: https://your-project.web.app/assets/logos/your-logo.svg
  • If deployed to GitHub Pages: https://username.github.io/repo-name/assets/logos/your-logo.svg
  • Upload to Firebase Storage (configure Firebase below)
  • Use your company website's logo URL

💡 Logo URL is saved automatically as you type
